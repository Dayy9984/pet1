# Pet Generate Benchmark

This repository is a neutral Next.js starting point for an LHP harness benchmark.

It intentionally does not include a product implementation, UI mock, prior Linear ticket list, or reference-service feature list. The harness should infer the product plan from the user-provided service link, reference image, and design-system source during a fresh planning session.

## Run

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Verify

```bash
npm run typecheck
npm run build
```
